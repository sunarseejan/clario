# clario
Automatically organize your files with smart detection and a modern GUI.

**Version:** 1.0.0

Clario is a desktop application that helps you automatically organize your files into categories such as Research Papers, Instructions, Images, Spreadsheets, and Archives. It includes **smart content detection** for PDFs and Word documents, a **modern dark-themed GUI**, and **real-time logs** of organized files.

---

## Features

- Automatic file organization by type
- Smart detection of:
  - PDFs as Research Papers
  - Word/TXT files as Instructions
- Dark theme GUI for comfortable viewing
- Real-time log of moved files
- Optional auto-run timer for background organization
- Version tracking
